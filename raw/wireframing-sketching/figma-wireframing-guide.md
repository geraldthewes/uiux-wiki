# What is Wireframing? The Complete Guide
## Figma Resource Library

**Summary**
This Figma resource provides a comprehensive guide to wireframing with practical checklists and best practices. It covers wireframing fundamentals, standard screen sizes, and guidance on low- vs. high-fidelity progression.

**Key Takeaways**
- 7 best practices for effective wireframing: identify goals, choose correct sizes, keep it simple, use conventions, focus on flows, annotate clearly, and iterate
- Standard screen sizes for mobile (393×852), tablet, and desktop breakpoints
- Guidance on low-fidelity vs. high-fidelity wireframes and when to use each
- Collaboration tips for working with teams using wireframes
- Free wireframing checklist included for practical application

**Origins/History**
Figma is a collaborative interface design tool that provides resources and educational content for designers, including comprehensive guides on wireframing best practices.

**Applications**
- Practical application of wireframing best practices
- Standardized approach to wireframe sizing and conventions
- Collaborative wireframing sessions with team members
- Using checklists to ensure wireframe quality and completeness
- Transitioning from wireframes to prototypes in Figma
