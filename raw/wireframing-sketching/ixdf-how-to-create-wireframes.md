# How to Create Wireframes: An Expert's Guide
## Interaction Design Foundation

**Summary**
This article from the Interaction Design Foundation outlines a comprehensive guide to creating wireframes, covering five core principles and the full wireframing process from research to iteration. It distinguishes between low, mid, and high-fidelity approaches and provides practical guidance on tools and techniques.

**Key Takeaways**
- Five core principles: simplicity, user-centricity, clarity, feedback loops, and consistency
- Full process: research → user flows → fidelity levels → iteration
- Distinction between low/mid/high-fidelity wireframes and when to use each
- Practical tools and techniques for effective wireframing
- Emphasis on iteration and user feedback throughout the process

**Origins/History**
The Interaction Design Foundation is an educational organization providing UX design courses and resources founded by experts in the field.

**Applications**
- Structured approach to wireframing for beginners and experienced designers
- Understanding when to use different fidelity levels
- Incorporating wireframing into a user-centered design process
- Creating wireframes that effectively communicate structure and functionality
