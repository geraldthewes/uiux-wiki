# Typography Basics in UI/UX Design

## Fundamental Principles

### Readability vs. Legibility
- **Legibility**: How easily individual characters can be distinguished
- **Readability**: How easily words, phrases, and blocks of text can be read

### Typographic Hierarchy
Establishes visual order and importance through:
- Size variations
- Weight variations (light, regular, bold, etc.)
- Case variations (uppercase, lowercase, small caps)
- Color and contrast
- Position and spacing

### Key Typographic Elements
1. **Typeface Selection**: Choosing appropriate fonts for context
2. **Font Size**: Base size for body text, scaling for headers
3. **Line Length (Measure)**: Optimal 45-75 characters per line for readability
4. **Line Height (Leading)**: Vertical space between lines (typically 1.4-1.6x font size)
5. **Letter Spacing (Tracking)**: Adjusting space between letters
6. **Kerning**: Adjusting space between specific letter pairs
7. **Alignment**: Left, right, center, justified

## Platform-Specific Guidelines

### Material Design Typography
- **Type Scale**: Defines a scalable set of styles
- **Roboto and Noto**: Default typefaces for Android and web
- **Dynamic Type**: Adapts to user preferences and screen sizes
- **Hierarchy**: Display, headline, title, subtitle, body, label, caption
- **Constraints**: Minimum touch target sizes for text-based controls

### Apple HIG Typography
- **System Fonts**: San Francisco (SF) for optimal legibility
- **Dynamic Type**: Supports user-preferred text sizes
- **Text Styles**: Large title, title, headline, subhead, body, callout, footnote, caption
- **Accessibility**: Supports bold text, increased contrast, and larger text options
- **Hierarchy**: Clear visual distinction between text styles

### NN/g Guidance on Typography
From NN/g Visual Design in UX study guide:
- Use typography to establish clear visual hierarchy
- Ensure sufficient contrast between text and background
- Limit typefaces to 2-3 families maximum per interface
- Consider reading patterns (F-pattern, Z-pattern) when placing text
- Use whitespace effectively to improve readability
- Align typography with brand voice and tone

## Best Practices

### Web Typography
- Use relative units (em, rem) for scalability
- Establish a consistent type scale
- Consider web font performance and loading strategies
- Test across devices and screen sizes
- Ensure accessibility compliance (WCAG)

### Mobile Typography
- Minimum touch target size: 44×44 pt (iOS), 48dp (Android)
- Consider thumb zones and reachability
- Use platform-native fonts when appropriate
- Adapt to different screen orientations and sizes
- Test legibility in various lighting conditions

### Font Pairing Principles
1. **Contrast**: Pair different classifications (serif with sans-serif)
2. **Mood**: Match the emotional tone of typefaces
3. **Hierarchy**: Ensure clear distinction between paired fonts
4. **Proportion**: Similar x-heights work well together
5. **Context**: Consider the content and purpose of the design
