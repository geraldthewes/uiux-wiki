# Interaction Design Foundation (IxDF) Overview

The Interaction Design Foundation provides educational resources on UX design, including literature, courses, and certificates.

## Key Offerings
- **Literature Library**: Extensive collection of UX textbooks and articles
- **Courses**: Self-paced and instructor-led courses on various UX topics
- **Certificates**: Industry-recognized certificates in UX design
- **Local Groups**: Community meetups in cities worldwide
- **Journal**: Peer-reviewed articles on interaction design

## Core Topics Covered
- UX Fundamentals
- Human-Computer Interaction (HCI)
- User Research Methods
- Information Architecture
- Interaction Design
- Visual Design
- Usability Testing
- Accessibility
- Mobile UX
- Service Design
- Design Thinking
- UX Management

## Learning Paths
- **Beginner**: UX Fundamentals, Design Thinking, User Research
- **Intermediate**: Interaction Design, Visual Design, Prototyping
- **Advanced**: Service Design, UX Strategy, Design Leadership
- **Specialized**: Mobile UX, Accessibility, Game UX, VR/AR UX
