# Color Palette Basics in UI/UX Design

## Fundamental Principles

### 60-30-10 Rule
A classic interior design rule adapted for UI design:
- 60% dominant color (primary/base)
- 30% secondary color
- 10% accent color

This creates visual harmony and hierarchy in interfaces.

### Color Harmony Types
1. **Complementary**: Colors opposite each other on color wheel
2. **Analogous**: Colors adjacent to each other on color wheel
3. **Triadic**: Three colors equally spaced on color wheel
4. **Monochromatic**: Variations in lightness and saturation of a single color

### Accessibility Considerations
- Minimum contrast ratio: 4.5:1 for normal text (WCAG AA)
- Minimum contrast ratio: 3:1 for large text and UI components
- Tools: WebAIM Contrast Checker, Stark plugin

## NN/g Guidance on Color
From NN/g article "Using Color to Enhance Your Design":
- Use color to draw attention to important elements
- Maintain consistency in color usage across the interface
- Consider cultural associations with colors
- Test color choices with real users, especially those with color vision deficiencies
- Use color to reinforce brand identity while ensuring usability

## Material Design Color System
- Primary, secondary, and tertiary colors
- Surface, background, and error colors
- On-primary, on-secondary, etc. for text/icons on colored surfaces
- Dynamic color theming capabilities
- 60-30-10 distribution implicitly supported through component usage

## Apple HIG Color Guidance
- Use system colors when possible for platform consistency
- Ensure sufficient contrast for accessibility
- Consider light and dark mode appearances
- Use color to communicate status (success, warning, error)
- Maintain visual harmony with platform conventions
