# Apple HIG Platform Guidelines

Apple's Human Interface Guidelines provide platform-specific recommendations for designing intuitive and consistent user experiences across all Apple devices.

## iOS
- **Navigation**: Use tab bars for peer content categories, navigation bars for hierarchical content
- **Adaptability**: Design for various screen sizes and orientations using Auto Layout and Size Classes
- **Interaction**: Leverage touch gestures (tap, swipe, pinch, etc.) appropriately
- **Feedback**: Provide immediate visual feedback for user actions
- **Typography**: Use Dynamic Type to respect user font size preferences

## iPadOS
- **Multitasking**: Support Split View, Slide Over, and Picture in Picture
- **Pointer Style**: Adapt UI for trackpad and mouse interactions
- **Landscape Focus**: Optimize for landscape orientation as primary usage pattern
- **Enhanced Interactions**: Utilize larger canvas for complex interactions and content manipulation

## macOS
- **Window Management**: Design resizable windows with appropriate chrome and controls
- **Menu Bar**: Utilize the menu bar for app-wide commands and status indicators
- **Keyboard Navigation**: Ensure full keyboard accessibility with logical tab ordering
- **Drag and Drop**: Support intuitive drag and drop interactions within and between apps
- **Touch Bar**: Provide contextual controls when available (on supported MacBook Pro models)

## watchOS
- **Glanceability**: Design for quick interactions lasting seconds
- **Navigation**: Use hierarchical or page-based navigation appropriate for small screen
- **Complications**: Create informative complications for watch faces
- **Haptic Feedback**: Use haptics to enhance user experience subtly
- **Digital Crown**: Leverage for scrolling, selection, and adjustment inputs

## tvOS
- **Focus Model**: Design for clear focus indication and navigation with remote
- **Top Shelf**: Utilize Top Shelf for dynamic content display in app launcher
- **Parallax**: Implement parallax effect for engaging UI elements
- **Menu Button**: Always allow return to previous screen with Menu button
- **Leanback Experience**: Optimize for 10-foot viewing distance and limited input methods
