# Apple HIG UI Elements

## Bars
- **Navigation Bar**: Contains buttons for navigating within a hierarchy of content
- **Status Bar**: Displays device and environment information
- **Toolbar**: Contains controls for performing actions on content
- **Tab Bar**: Enables switching between different subtasks, views, or modes

## Views
- **Content View**: Displays app-specific content
- **Collection View**: Presents data in customizable layouts
- **Table View**: Displays a list of items in a single column
- **Split View**: Divides screen into two or more resizable panes

## Controls
- **Button**: Initiates an app-specific action
- **Switch**: Toggles between two states
- **Slider**: Selects a single value from a continuous range
- **Picker**: Displays one or more components that users can rotate to select values
- **Page Control**: Indicates which page is currently visible in a sequence

## Temporary Views
- **Alert**: Provides important information requiring user confirmation
- **Action Sheet**: Displays a set of alternatives related to a task
- **Activity View**: Shares content with other services or copies to clipboard
- **Share Sheet**: Shares content with other services or people

## Extensions
- **Today Widget**: Displays timely, relevant information in Today view
- **Share Extension**: Shares content with other services
- **Action Extension**: Manipulates or transforms content within host app
