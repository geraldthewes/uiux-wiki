# Visual Design in UX - NN/g Study Guide Summary

## Key Principles

### Visual Hierarchy
- Use size, color, contrast, alignment, and spacing to guide attention
- Most important elements should be most prominent
- Establish clear scanning patterns (F-pattern, Z-pattern)

### Consistency and Standards
- Maintain internal consistency (within product)
- Follow external consistency (platform conventions)
- Use established UI patterns and controls

### Feedback and Response
- Provide immediate feedback for user actions
- Use microinteractions to enhance usability
- Ensure feedback is clear and noticeable

### Accessibility
- Ensure sufficient color contrast (4.5:1 for normal text)
- Use scalable typography
- Provide alternative text for non-text content
- Design for keyboard navigation

### Typography
- Limit typefaces to 2-3 families
- Establish clear typographic hierarchy
- Ensure readability and legibility
- Consider line length and spacing

### Color Usage
- Use color to convey meaning and status
- Maintain consistency in color application
- Consider cultural associations with colors
- Test for color blindness

### Imagery and Icons
- Use meaningful, relevant images
- Ensure icons are recognizable and clear
- Maintain consistent icon style
- Optimize images for performance

### Layout and Spacing
- Use grids for alignment and consistency
- Apply appropriate white space (padding, margins)
- Group related elements using proximity
- Balance density and openness

### Motion and Animation
- Use animation purposefully to guide attention
- Keep animations brief and non-distracting
- Follow platform motion guidelines
- Consider performance implications
